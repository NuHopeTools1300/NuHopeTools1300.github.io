NuHopeTools / Workflow Architecture

Research Workbench System Map

This is the visual end-state map: not a redesign of one legacy tool, but a set of interoperable research surfaces built on one shared object model. The system must support both evidence-first and entity-first work, with reliable pivots between images, regions, kits, placements, locations, claims, sources, and events.

What Each Surface Owns

Images

Inspection, annotation, image-region correction, entity linking, claim formation.

best at visual evidence-first work

Entities

Start from kits, parts, placements, or objects and pivot outward into every linked support and conflict.

best at reverse lookup

Maps

Location review, map-area pivots, placement inspection, manual position overrides.

best at spatial work

Sources

Non-Negotiable Pivots

image → regions region → entity region → claim entity → linked regions entity → linked images location → linked evidence placement → map position source extract → claim event → evidence

Simple test: if a user can enter the system from one side but cannot pivot back out through the linked objects, the surface is incomplete.
