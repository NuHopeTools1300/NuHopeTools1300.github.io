Decals,.. OH BOY!

I did the latest version available (YT-1300 Corellian Freighter DECAL SET  by ACME™ Inc. MPC/AMT) with a massive amount of research from people on this board.  There are 200 some odd decals on the original 5 ft. falcon. Some 400 IF you include the Hash marks.

I have been asked if I would create a set for the 5 foot a few times. I have always said "not so much". When you scale things up you would be surprised at all the flaws that become apparent. And It is more work than I have time for as time spent that can't be recouped financially. So there IS that.

If I was building a 5 ft falcon I would do the same thing your thinking about, I e using original decals taken from the donor kits as much as possible.  Which IS what ILM did, SOOOooo,..

To date the http://hamptonroadsscalemodelers.com...filming-model/ web site is probably the most comprehensive site concerning decals at this time if you ask me.

When you get to that point if you have any problems with reference let me know and I will see what I have concerning same.
