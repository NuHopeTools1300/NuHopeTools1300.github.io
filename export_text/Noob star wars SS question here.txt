I think you'll find that studio scale is not really about building a kit - although sometimes there are kits made from the efforts of studio scale modelers.  If you're looking for studio scale kits you'll need to do a bit of research - this site is a good place to start...  I'd start in the junkyard and projects section.

If you really want to get into studio scale modeling - you'll find help here as well but it is essentially about trying to replicate as exactly as possible the original studio miniature.  Where possible using the same techniques and parts...  much different that buying a kit that happens to be studio scale...  but its easy to see where a new person gets confused about it.
