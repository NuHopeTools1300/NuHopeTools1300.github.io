the star wars chronicles book has all the reference angles you would need of the actual filming miniature. I have a few larger versions of those pics as well that I found on the web.  try googleing through there isn't a lot of info out there but a heck of a lot more than some other ST references like the fuel tank for example, tried for years to find good reference of it but nothing has surfaced yet.  

If your going for a true Studio scale Starspeeder, make sure you see that there are two different models out there, the original used for the filming, and another one made later for use as a scale reference during the Tokyo ST construction.  This same Tokyo model was cast and painted for the starspeeder 3000 scheme model that was previewed in Disneyland.  There are a lot of different details on the Tokyo one compared to the filming miniature, especially the details on the engine section in the back, as well as the overall size.
