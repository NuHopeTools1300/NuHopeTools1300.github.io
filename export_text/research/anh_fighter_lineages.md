# ANH Fighter Miniature Lineage Crosswalk

## Scope

- Film scope: `Star Wars: Episode IV - A New Hope` only.
- Physical scope: real `ILM` miniatures and immediate pre-production/reference builds tied to `ANH`.
- This document separates `physical lineage` from `identity state`.

## Working Rule

- A `physical lineage` is one build/object when the evidence supports that level, or one object family when the evidence does not.
- An `identity state` is a paint, marking, or screen-assignment state worn by that object.
- A name like `Red 3`, `Gold Leader`, or `Pentagon TIE` is therefore not automatically a separate physical miniature.

## Confidence Guide

- `High`: directly supported by unusually strong period or archive evidence.
- `Medium`: likely correct, but still partly inferential.
- `Low`: useful working hypothesis only.

### X-Wing Read

- `X-Wings` are now good enough for object-level lineage work.
- The key discipline is to separate `identity label` from `physical object`.
- The main trap is assuming that every `Red 3`, `Red 4`, or `Red 10` mention refers to a unique fully separate miniature.

### Y-Wing Read

- `Gold Leader / TIE Killer` is the hardest anchor.
- The live conflict is not whether `Tiger Sprockets` is real; it is how exactly its screen role overlaps with `Gold Leader / TIE Killer`.
- The `red stripe blue artoo` / `Magic of Myth` object is definitely a single preserved object.
- What remains open is whether that object is its own ANH survivor or whether it collapses onto another already-known preserved Y-wing lineage.

### TIE Read

- `TIEs` remain better described as `lineages plus states` than as individually named ships.
- The biggest gain here is methodological: `Pentagon TIE` is more defensible as a state label than as a separate object.
- The `Chronicles TIE` and `Edlund damaged-wing TIE` are the only ANH TIE cases in this set that currently feel object-like rather than merely state-like.

## Bottom Line

- `X-Wings`: strong enough now for a precise lineage/state split.
- `Y-Wings`: strong enough for a serious working table, but not yet for a fully closed object census.
- `TIE Fighters`: strong enough for a disciplined state table and a smaller number of object-like lineages.
