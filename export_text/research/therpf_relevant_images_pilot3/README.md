# therpf relevant images pilot

This folder is a selective pilot extraction from the mirrored therpf archive at:

- `D:\Manuel\MillenniumFalcon\Websites\therpf_mirror`

It is meant to be a browsable, deduplicated working set rather than a full import.

Start here:

- `index.html` - thread index and entry point
- `thread_manifest.csv` - one row per selected thread
- `image_manifest.csv` - one row per copied unique image
- `image_links.csv` - link table from thread/post to image
- `thread_texts/` - copied thread text exports when a matching `.txt` was found
- `thread_pages/` - per-thread gallery pages
- `images/` - deduplicated copied image files

Current pilot scope:

- `20` selected strong-tier threads
- `981` copied unique images
- `1146` linked image occurrences
- `662` missing image references from the mirror
- `15` selected threads with copied text exports

Generated by:

- `export_text/research/scripts/extract_relevant_therpf_images.py`
