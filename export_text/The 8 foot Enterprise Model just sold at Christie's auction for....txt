Kurt,

There is a thread on this model here:

http://www.starshipbuilder.com/forums/showthread.php?t=1193

It's not really the "Phase II" as that project was started and never finished. Word has it the actual Phase II model was built by Don Loos (IIRC). This model was built by Brick Price. It basically looks like the TMP final design but with the Phase II nacelles, so it is a strange "hybrid" miniature. It's about 4 feet long, or 1/2 the size of the final TMP filming model.
