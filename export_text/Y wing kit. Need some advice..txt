For a complete kit and being highly accurate the Nice-N/Salzo version is the easiest way to go as said before. You have to be persistent in getting ahold of Mike. I've found better luck on Facebook.

Or you can go the DaveG route! The Y-Wing "Green Leader" Project - NOW OPEN SOURCE!

This is not a kit, but a start using 3D files. You are left to source the parts which is hard, but really rewarding. This was the most fun I've ever had building anything. There are a lot of places to go to source many of the common nurnies of each Y.
