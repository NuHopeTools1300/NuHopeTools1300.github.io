Thanks, yes I was aware of the hull thickness and how almost impractically thin it is.  I did make the decision to use 3mm to make it.  I don't think i would mold something this big but wanted to build it out of something that i would at least have the option of doing, 1.5mm would make that impossible IMO.  At 3mm it still looks correct (IMO) to the scale of 1/6, meaning while it looks thick compared to the studio model it does not look wacky for the 1/6 scale.

I know it's one of those things that can get controversial when looking at the accuracy angle, but I guess i chose to err on the side of sturdiness on this one.  It was not an easy choice.

I can also trick out the edge a bit if I really want to and pretty easily gently taper the edge down to say 2mm.  It would be a trick of the eye but would work.

If i do mold and cast it there would be some compression going on as well, thinning the entire fuselage.

So, not a lot of time to do much today because I've been shoveling for 4hrs so far today.  That's right 3 feet of snow.

So this is the part of the speeder that gets pretty dicey, (and pricey with this renshape).  Once the bottom is done i think this will start to be a bit more fun, right now it's stressful.  The bottom has these weird runners that go the length of the ship,  it's really the only part of the ship that looks "sculpted"  I am sure there is a few different ways to do this but the middle bottom plate has 3 alignment pins (2 visible from this side) that lock it into the top via the floor of the cockpit.  It's essential to have these pins secured in as i build the bottom so everything stays where it should, and if i kit this it will help people assemble it as opposed to "guessing" where the body lines up.  The bottom has just a slight curve up at the nose right at the point that the cockpit ends, so i needed to clamp all this together as i glued up the renshape.  

I figure that this step by step may help anyone who wants to build one of these for themselves.  After clamping up the floor i mark out where the inner wall of these runners need to go. Then the renshape panels are cut at the max dimension for the panels at the back.  Those are then glued into place while the whole thing is clamped up with the curve in place.  I needed to shim the parts to secure up that curve.  There is a lot of excess on these now but will get to work on them with a rasp tomorrow.  The great thing about the renshape is that when this is all shaped it will require only a small amount of apoxy sculpt to finish them.

I wanted to use modeling foam for this part as it is much easier to sculpt, but because these runners are really the connective tissue for the rest of the panels in needed to be renshape, which will accept pins and glue much easier.

If i was not building this to be molded i could get away with using styrene for this part, but felt that this is a safe move, if not more work in some ways.  But the renshape is very forgiving if i need to make adjustments.

thanks

mike

Not much of an update since i am so busy with shuttle molds but here is a little bit.  Hope this can help anyone who may want to do the airbrakes without the arduino board.  This was way more simple than i thought, so far.  The controller is a $40 special that comes with the controller and the receiver and is 6 channels.  Once bound to the controller simply plug in the battery and the servos and they work!  I can plug in the controller to the PC and even do some programing if needed.  Over all it was cheap and pretty easy.  $40 for the controller, $10 for each servo, $20 for the battery, not bad.  This should work on the smaller SS one as well as long as the servo's you get fit.
