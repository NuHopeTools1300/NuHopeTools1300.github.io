Well I' ve started my first X wing build in about thirty years. I finally managed to get a Salzo V3 and I have to say its about as near perfect as I imagined it would be after seeing all the excellent X wing prop versions built of it here and elsewhere.  It really gave me an odd kind of thrill to see it all laid out and ready to go.The casting quality  meant there has been little to clean up as well as it fits together cleanly and nicely.

   Having looked at so many builds and photos I've decided to go my own way and not go for strict accuracy to one example. I've used the 4ft X wing exhibition model as a template for added detail, particularly around the engines , which  has always impressed me ( though I don't think this was ever used in filming). I just liked the detail and it seemed a bit different to everyone else versions, and the kit leaves you damn little to do as is ,as  it is that accurate to the screen version of the original X wing.

   So the painting will be free lance, I guess Rogue leader (I liked playing that game alot) though pretty close to the exhibit model. it should get done over the next couple of weeks.

Well ,I'm back on the X wing now that the crawler is done. I've finished all the electronics in the cockpit and the engines and have put the main body together.

    I've made the nose like the 4ft version with the cut towards the front. It ment I had to extend it by about 3mm but it seems to look alright. If anyone thinks otherwise then let me know ,but it will have to be quick as I start  the finishing paintwork from Wednesday. 

  I like the engines now they have been done and dirtied, much more like I hoped. All the wings, lazers and body are assembled. Mike ,thanks for the info on the snow shovel part, when I saw where it was to supposed to go I laught, I couldn't believe I missed it!!!

   I will be using no set template for this X wing, though it will closely resemble the 4ft version. I've built the cockpit like Wedges RTOJ version because the 4ft version one doesn't seem like an X wing one proper.

 Overall I like alot of the painted detail on it ,but I will alter some of it, as there are weathering details on the studio full sized ones I want to include. It will be a bit slow as drying times in the cold weather slow things up. I simply want to produce an X wing that looks as good as I can get it, and ,as I've said before all the  versions I know of have already been done very impressively, so theres no need to reproduce them again exactly. So I guess it will be a bit of a Rogue then!
