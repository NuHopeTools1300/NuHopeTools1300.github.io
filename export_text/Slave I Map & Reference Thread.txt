Hey Everyone,

One of my top modeling targets is Slave I. I've been slowly gathering parts and mapping the nurnies but could use the communities help knocking out some IDs.

First up is the underside/engine bay. Because this area is so dense with parts, I've separated each map by brand, and each brand has a color that will remain consistent for all maps. See the last image for numbered parts in purple that I have not been able to ID - any help is greatly appreciated. Once we get these knocked out, I'll move on to the hull and then the cockpit.

Special thanks to rooker2k  aka Marcel Leuthold from Scifi Studio Scale Modellers , Studio Kitbash, GEEK1138 , and sjanish for their help getting to this point!
