After a long time I've finally decided to take the plunge and built a model. I think the LR 232 is a nice model to start my model building journey. I've read a lot of threads on several forums so I think I know what I'm getting into. The only thing I'm afraid of is getting addicted :lol.

First of all, I wanna thank all the guys that idenitified all the parts they used for the pod. They deserve all the credits, I'm just following in their footsteps.

I'm gonna document everything step by step so I hope this thread will be a great help to future (newbie) modelbuilders.

From what I've read I'm gonna need these kits:

1/72 Hasegawa Morser Karl

1/35 Tamiya Schwerer Panzerspahwagen Sdkfz 232 (8Rad)

1/35 Tamiya 20mm German Flakvierling 38 MIT Sd.Ah. 52

1/25 Tamiya Russian Medium Tank T-34 Type 85

1/12 Tamiya Lotus JPS MK III

Plastruct LF-416

Plastruct VHE-40

Right now the only kit I've got is the Tamiya Flak 38. Several more are on the way though. If you wanna use all original parts you'll need 5 kits. I've studied pics of the model and the kit itself and I think these parts are the ones I need:

A3, A5, A9, A10, A13, 

E10, E11

F17
