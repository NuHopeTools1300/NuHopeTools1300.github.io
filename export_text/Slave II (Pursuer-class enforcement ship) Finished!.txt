With the "Hero" TIE Bomber finished for our film:

http://www.therpf.com/f11/tie-bomber-amt-tie-advanced-x-1-now-studio-scale-212845/index2.html

I've been able to start work on the antagonist's vessel. I'm using a version of Slave II, one of Boba fett's ships from the EU. The reason behind this choice is simplicity and cost - the vessel is entirely made of flat surfaces, and onlky a few of those, so it's possible to make a decent model from scrap in a short period of time. So far foamcore (sintra board) and plasticard have seen use to make the main fuselage. I'll be fabricating the wing and greeblies tommorow.

[IMAGE: 7DgnkJ5.jpg]

[IMAGE: 49l4Cqp.jpg]

Filming for the effects shots commences on wednesday, i'll post updates here as the build progresses. 

Boba Fett won't be featuring, for reference - this is intended as a pirate ship of sorts.

Thanks! I put in it here as it's a shooting model for our own film/studio, but I don't mind where it goes really.

Progress wise, i'm about halfway through painting now. Bit of a rush job, but we're hoping to shoot on wednesday and I need to sort the digital backgrounds as well. I'm using a tan basecoat, followed by some pre-shading and a light tan top coat + diffusion weathering. I wanted to use a pale green as per wookiepedia, but we're using a green backdrop for the chroma key!
