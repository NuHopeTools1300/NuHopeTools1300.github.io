Dave,

If you already have the models to build an original Falcon, then you don't need a 3D printed Y-Wing, as you've also by default already got the models to build the Y-Wing.  So just build the armature, and start adding original greeblies.  The venn diagram overlap among the kits, and which props they produce, is half the fun of studio scale, and I started with the Y-Wing, and then only retroactively realized, "Hey, if I've got these kits already, then I only need a few more to build a Falcon."  So if you're starting with the Falcon, there's very few you don't already have to build a Y-Wing.

Thanks.

I would point out that layer lines are far more printer dependent than source file dependent. Sure, I've seen stl's that have spheres looking more like D&D dice than true spheres, but that still has nothing to do with layer lines showing up because it was printed on a low end resin or a FDM printer. I have a 14k resin printer with a 0.01mm layer height. Part orientation on the platform also helps with reduction in layer lines. Layer lines are pretty much invisible on my prints with any sanding required probably less than I get from off the shelf plastic model kits.

I'm aware of the kits involved for scratch building, which is why I was looking for a complete 3D printable source that would allow me to replace some of the major assemblies with parts from kits I already have. Many of the greebly sub sections were also used on the 4 foot Falcon, so I have quite a lot of the kits and part assemblies already. Those that I don't have, or that I don't feel like fussing with, that's what the 3d print would take care of.

It's why I was looking for an accurate one. I'm sure I could scratch build one if I wanted to put that much effort into it. But this is a side quest project. I just don't want to go through the whole 'trying to find accurate resources and drawings' again like I'm doing with my Falcon build. I was looking for a solid starting point to work from.

I have the Cowley Y wing and it’s more than 500 stl’s where you can print the ship in chunks and the biggest piece is the front piece and you will need a printer that can do at least 210mm. I’m gonna print mine on a Anycubic photon mono 4 ultra 10k resin printer for the body parts and Greeblies for the blank model and the finished parts as well.

 It goes together sort of like an older resin kit where 90% of the greeblies are already on the model and you put on the fine detail parts. I’m getting larger resin printer that can print 300mm long  items so I can print a x wing fuselage top and bottom and at 300mm x 298mm x 164mm print area with the Anycubic photon mono 7 max there isn’t a whole lot I can’t print.  

It’s cheaper for me to just print the models I want to build as opposed to getting Shapeways or the like to print for me. But there aren’t any directions or anything the Greeblies aren’t named so you can have fun opening each file with a stl viewer and id’ing each one and naming it as such. You can do things several ways. Use the Greeblies from the cowley y wing and put them on the blank body parts and armature from Dave G’s open source project. Or you can use Cowley’s blank body parts and engine nacelles and his greeblies  and put one together that way. Or you can print out the 90% done model and add the fine detail greeblies provided. It’s more than 2gb of files and I’m gonna print out the 90% finished ship out first. It fits together really solidly with no armature if you go the Cowley body parts route. Then I’m gonna build a ship and put it together   happy to be able to finally move forwards and build and paint a studio scale Y wing. Hope this helps a little.
