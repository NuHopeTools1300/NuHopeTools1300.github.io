# Kit Taxonomy

This project uses a two-level kit taxonomy:

- `category_family`: broad domain
- `category_subject`: more specific subject within that domain

The current families are:

- `aircraft`
- `military_ground`
- `automotive`
- `commercial_vehicle`
- `naval`
- `space`
- `industrial`
- `other`

The current subjects include:

- `military_aircraft`
- `civil_aircraft`
- `helicopter`
- `afv`
- `artillery`
- `rail_artillery`
- `support_vehicle`
- `motorcycle`
- `car`
- `race_car`
- `truck`
- `trailer`
- `warship`
- `battleship`
- `carrier`
- `destroyer`
- `submarine`
- `small_craft`
- `undersea`
- `spacecraft`
- `rocket`
- `component`
- `structure`
- `unclassified`

This is intentionally practical rather than academically pure. It is meant to make the donor-kit corpus easier to browse, filter, and compare, while staying close to familiar model-subject groupings used by sites such as Scalemates.
